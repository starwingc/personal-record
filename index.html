饮食与训练指南

核心原则

已经规律训练 4 年、腹肌轮廓可见、大腿相对偏瘦 —— 这不是从零开始的增肌减脂阶段,而是"精修阶段"。目标是小幅度、可持续的体脂下降,避免流失来之不易的肌肉。以下建议都不需要知道体重,用相对量和习惯来把握。

  • 每餐优先蛋白质,用手掌大小估份量,而不是称重记克数。
  • 力量训练即使只有 5lb,也可以靠"增加次数/组数、放慢离心、缩短组间休息、循环训练"持续制造训练量增长,不必靠加重量进阶。
  • 饮食用餐盘法:半盘蔬菜、1/4 蛋白质、1/4 复合碳水;减少精加工食品和添加糖;把每餐分量精简 10-15% 来制造温和的热量缺口,而不是严格计数卡路里。
  • 保证纤维、饮水、睡眠(7-8 小时)——睡眠直接影响激素和恢复,对"减脂不掉肌肉"很关键。
  • 已有的大量有氧(隔天 90-120 分钟踏步机)不建议再加量,避免影响肌肉保留;可以在踏步的同时用 5lb 哑铃穿插做侧平举 / 前平举 / 竖直上举 / 前平举内收。

早餐 / 晚餐份量参考(小盒子 / 拳头估算)

不用称重,按你平时备餐用的小盒子或者自己的拳头估算就行:

  • 早餐:蛋白质约 1 拳头(2个鸡蛋 / 1杯希腊酸奶 / 1盒鸡胸或豆腐),主食约 1 拳头(燕麦、全麦吐司、杂粮),再加一点蔬果;用小盒子装大概 半盒到 2/3 盒。
  • 晚餐:蛋白质约 1 拳头,主食比早餐略少、约 半拳头到 1 拳头,剩下的空间尽量装满蔬菜(至少半盒);用小盒子大概装 1 盒左右。
  • 午餐没有单独列,原则一样:半盒蔬菜、1/4 蛋白质、1/4 主食,按前面的餐盘法估算。

结合生理周期调整

  • 卵泡期(经期结束到排卵前):精力通常较好,力量日可以适当增加强度或组数。
  • 黄体期(排卵后到下次经期):食欲增加、水分潴留是正常现象,可以适当增加复合碳水,不必因体重/围度波动焦虑;感觉疲惫时多用"顺延"功能。
  • 月经期:强度可以主动降低,多吃含铁食物,这个阶段可以放心多用"顺延"。

周期阶段为估算,非医学诊断,身体感受始终优先。

6 个月分期

  • 第 1-2 月:建立习惯与打卡一致性,重点是蛋白质优先和坚持记录。
  • 第 3-4 月:适度收紧饮食质量与分量,力量日增加次数/组数/循环强度。
  • 第 5-6 月:通过照片、围度、衣服合身度(而不是只看体重)来复盘和微调。

训练动作说明

有氧日(踏步机 90-120 分钟):每 15-20 分钟穿插一组哑铃动作,每个动作 30-45 秒:侧平举、前平举、竖直上举、前平举内收。

力量日(20-30 分钟,徒手 + 2×5lb):

  • 深蹲3组 x 15次
  • 臀桥 / 顶髋3组 x 15次
  • 后撤箭步蹲(左右各)3组 x 10次
  • 侧卧抬腿(左右各)2组 x 15次
  • 驴踢腿(左右各)2组 x 15次
  • 提踵3组 x 20次
  • 平板支撑3组 x 30-45秒
  • 死虫式2组 x 12次
  • 侧平举 / 前平举 / 竖直上举 / 前平举内收(5lb)各 3组 x 12-15次
  • 俯卧撑3组 x 力竭前
